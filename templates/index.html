How it works

From GitHub profile to CS player card in three steps

1

Fetch

GitCSTV pulls your public GitHub profile, repositories, and recent activity via the GitHub REST API — no authentication required.

2

Analyze

Stars, forks, followers, languages, push frequency, and account age are scored across six dimensions to produce a CS-style rating.

3

Generate

A high-resolution HLTV-style player card is rendered with your avatar, stats bars, awards, and rank — ready to download and share.

Scoring dimensions

Stat What it measures Driven by
FIREPOWER Raw code output & productivity Repo count, push frequency, creation events
STARPOWER Star impact & project popularity Total / max / average stars per repo
LONGEVITY Account & project staying power Account age, maintained repos ratio
IMPACT Community reach & influence Followers, forks, watch events, PRs
COLLAB Open-source collaboration Language diversity, PRs, issues, forks
VERSATILE Technology range & breadth Language count, topics, repo diversity
How MVP, EVP, and Events are assigned:
Repos with ≥1,000 stars earn MVP awards. Repos with ≥100 stars earn EVP awards.
Event placings rank your top repositories by star count — your "tournament results" on GitHub.

Built with Python, Flask, Pillow, and the GitHub REST API. Cards are rendered in HD (800×1150px PNG) and delivered as base64 for instant preview.
